## How to run:

### Step 1:
```
Download from Canvas
```
### Step 2:

```
Extract T-113-VLN1-H30-main.zip
```
### Step 3:
```
Open Command prompt (Windows) or Terminal (Mac/Linux)

```

#### Windows
